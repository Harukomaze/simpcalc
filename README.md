# simpcalc
A pip package which uses math.js api for calculations

```py
pip install simpcalci
```

# Usage
```py
from simpcalc import simpcalc

calculator = simpcalc.Calculate()


print(await calculator.calculate('1 + 2 * 2 / 2 ** 2 ^ 2 + sqrt(3) + (7*56)'))

...
>>> 394.9820508075689
```
